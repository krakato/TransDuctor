🌐 Página de Prueba - TransDuctor

Usa esta página para probar la extensión. Pasa el cursor sobre cualquier texto.

✅ La extensión está correctamente instalada si ves tooltips de traducción al pasar el cursor.

📝 Prueba 1: Palabras en Inglés

Pasa el cursor sobre estas palabras para ver su traducción:

hello • world • translator • extension • computer

📖 Prueba 2: Oraciones Completas

Intenta pasar el cursor sobre estas frases:

"Hola mundo, esto es una prueba de traducción."

"The quick brown fox jumps over the lazy dog."

"Bonjour, comment allez-vous?"

🌍 Prueba 3: Múltiples Idiomas

Prueba con textos en diferentes idiomas (cambia el idioma destino en opciones):

  • Inglés: "The extension is working perfectly!"
  • Español: "¡La extensión funciona perfectamente!"
  • Francés: "L'extension fonctionne parfaitement!"
  • Alemán: "Die Erweiterung funktioniert perfekt!"
  • Japonés: "拡張機能は完璧に機能しています！"

⚙️ Prueba 4: Cambiar Configuración

Prueba estos ajustes para personalizar la experiencia:

  • Aumenta el tiempo de espera a 3000ms para ver si prefieres esperar más
  • Cambia el tamaño de fuente a 14px para texto más grande
  • Cambia el tema a claro si usas fondos claros
  • Prueba diferentes idiomas destino

🔍 Prueba 5: Verificar en Consola

Para debugging, abre la consola del navegador (F12) y verifica:

Mensajes que deberías ver: ✓ TransDuctor iniciado correctamente [Logs de traducción]
